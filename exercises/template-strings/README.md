# Template strings

## Multi-line

Należy do zmiennej `description` przypisać podany tekst

```
Ala
ma 
kota,
a kot ma Alę
```

## Interpolacja
Należy odpowiednio zadeklarować zmienną `msg`, aby otrzymać 'Mam na imię John, mam 10 lat'.
```javascript
const name = 'John';
const age = function() {
  return 10;
};

```
